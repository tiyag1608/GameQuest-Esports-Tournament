# 🎮 GameQuest – Esports Tournament Management System

## 📌 Project Overview

GameQuest is a web-based platform designed to manage and display esports tournaments.
The system allows users to view tournaments, track match brackets, and manage tournament information through a simple and interactive interface.

This project demonstrates how basic web technologies can be used to create a functional esports tournament management platform.

---

## 🚀 Features

* Tournament listing and overview
* Interactive tournament dashboard
* Bracket visualization for matches
* Clean and responsive user interface
* Easy navigation between pages

---

## 🛠 Technologies Used

* **HTML** – Structure of the website
* **CSS** – Styling and layout design
* **JavaScript** – Interactive functionality

## ⚙️ How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Run **index.html** in any web browser.

The website will load and you can navigate between the different pages.
